# Quick Code Review

다음 기준으로 코드 리뷰해줘:

## 체크리스트
1. **버그 가능성** - 잠재적 런타임 에러, 엣지 케이스
2. **보안** - 인젝션, 민감정보 노출
3. **성능** - 불필요한 연산, N+1 쿼리
4. **가독성** - 네이밍, 함수 길이, 복잡도

## 출력 형식
```
✅ 좋은 점: ...
⚠️ 개선 필요: ...
🐛 버그 위험: ...
```

한국어로 간결하게 답변해줘.
